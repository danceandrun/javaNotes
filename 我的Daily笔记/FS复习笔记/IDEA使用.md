# 快捷键

|         快捷键         | 描述                     |
| :--------------------: | :----------------------- |
|       ctrl + F12       | 查看类中所有方法的快捷键 |
|    alt + shift  + ↑    | 快速移动一行代码         |
|        ctrl + w        | 快读选中当前内容         |
| 🎆CTRL + shift + enter  | 快速补全代码             |
|     shift + enter      | 另起一行                 |
|    CTRL + shift + V    | 查看历史粘贴板           |
|    CTRL + shift + F    | 全局搜索，搜索内容       |
|       🎆ctrl + n        | 找类和接口               |
|       双击shift        | 搜索文件                 |
|     CTRL + alt + O     | 删除无用的包             |
|        CTRL + H        | 类的继承关系             |
| Ctrl + alt + shift + J | 将相同内容都选中         |
